# Assembly Line Bottleneck Diagnosis Simulator

A configurable manufacturing simulation and bottleneck diagnosis tool built using SimPy and Streamlit.

## Features

- Configurable production lines
- Process and assembly stations
- Buffers and WIP tracking
- Bottleneck diagnosis
- Scenario analysis
- Streamlit dashboard

## Tech Stack

- Python
- SimPy
- Pandas
- Plotly
- Streamlit

## Run

```bash
streamlit run app.py
